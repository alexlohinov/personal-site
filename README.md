# Alex Lohinov

Personal portfolio built with Next.js.

## Stack

- Next.js
- TypeScript
- Tailwind CSS
- MDX
- next-themes
- Lucide React
- Vercel Analytics

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

Designed to be deployed on Vercel.

## Notes

The website intentionally stays small, minimal and content-focused.

Writing is stored as MDX and opens directly from the homepage.
